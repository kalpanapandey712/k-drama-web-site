How to Run This Project

  1. Save this file as signup.html in your kdrama-otp folder
  2. Make sure you have Node.js installed on your computer
  3. Open terminal in your project folder and run:
    • npm init -y
    • npm install express dotenv bcrypt body-parser
  4. Create a server.js file with your backend code
  5. Run your server with node server.js
  6. Open your browser and go to http://localhost:3000
